⚙️ Configurar Backend
El backend (Google Apps Script) hace las consultas de forma segura.
Tu token nunca se expone al navegador.
1Crear el proyecto
Ve a script.google.com → Nuevo proyecto
Pega el contenido del archivo BACKEND_GoogleAppsScript.js que descargaste
Agrega tu token en la variable PERSONAL_TOKEN
2Publicar como Web App
Implementar → Nueva implementación → Tipo: Aplicación web
Ejecutar como: Yo (tu cuenta)
Quién puede acceder: Cualquier usuario
Clic en Implementar → copia la URL
3Pegar la URL aquí
La URL tiene este formato:
https://script.google.com/macros/s/XXXX.../exec
